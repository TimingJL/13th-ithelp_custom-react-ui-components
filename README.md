# 30 天擁有一套自己手刻的 React UI 元件庫

## Installation

**npm**

```shell
npm i 13th-ithelp_custom-react-ui-components
```

**yarn**

```shell
yarn add 13th-ithelp_custom-react-ui-components
```

## Usage

```javascript
import { Button } from '13th-ithelp_custom-react-ui-components';

function App() {
  return (
    <div className="App">
      <Button>Submit</Button>
    </div>
  );
}

export default App;
```
